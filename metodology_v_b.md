# Методология оценки: Версия B (5 критериев, расширенная)

## Общая информация

- **Курс:** Продуктовый и диджитал-маркетинг
- **Версия:** B — 5 критериев, 20 подаспектов (расширенная)
- **Шкала:** 1-10 по каждому подаспекту
- **Итоговая оценка:** взвешенная средняя по 5 критериям
- **Порог "не зачтено":** средний балл < 4.0 ИЛИ любой из 5 критериев < 3
- **Когда использовать:** для более детальной оценки, итоговых работ или когда нужна дифференциация между сильными работами

## К4. Аналитическая глубина (20%)

Оценивает уровень аналитического мышления и глубину проработки.

### 4a. Обоснование выбора бренда/продукта

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Бренд выбран без объяснения |
| 3-4 | Не зачтено | Формальное обоснование ("нам нравится") |
| 5-6 | Удовлетворительно | Есть обоснование, но поверхностное |
| 7-8 | Хорошо | Обоснование с учётом маркетинговой ситуации бренда |
| 9-10 | Отлично | Глубокий анализ рыночной ситуации бренда + обоснование выбора |

### 4b. Анализ конкурентной среды

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Конкурентная среда не рассмотрена |
| 3-4 | Не зачтено | Упоминание 1-2 конкурентов без анализа |
| 5-6 | Удовлетворительно | Перечисление конкурентов с базовым сравнением |
| 7-8 | Хорошо | Анализ конкурентов с выводами для маркетинговой стратегии |
| 9-10 | Отлично | Глубокий конкурентный анализ + позиционирование + дифференциация |

### 4c. Понимание целевой аудитории

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | ЦА не определена |
| 3-4 | Не зачтено | Общее описание ("молодёжь 18-35") |
| 5-6 | Удовлетворительно | Сегментация по базовым демографическим признакам |
| 7-8 | Хорошо | Детальные портреты сегментов с психографикой |
| 9-10 | Отлично | Портреты + поведенческие паттерны + customer journey |

### 4d. Реалистичность предложений

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Предложения нереалистичны или фантастичны |
| 3-4 | Не зачтено | Шаблонные предложения без учёта специфики бренда |
| 5-6 | Удовлетворительно | В целом реалистичные, но без учёта ограничений |
| 7-8 | Хорошо | Реалистичные предложения с учётом бюджета и ресурсов |
| 9-10 | Отлично | Реалистичные + с примерами из практики + учёт рисков |

## К5. Качество презентации (15%)

Оценивает качество оформления и подачи материала.

### 5a. Структура и логика изложения

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Хаотичная подача, нет структуры |
| 3-4 | Не зачтено | Есть разделы, но логика не прослеживается |
| 5-6 | Удовлетворительно | Базовая структура, но перескоки между темами |
| 7-8 | Хорошо | Чёткая структура, логичная последовательность |
| 9-10 | Отлично | Безупречная структура, каждый слайд логично вытекает из предыдущего |

### 5b. Визуализация данных

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Только текст, нет визуализации |
| 3-4 | Не зачтено | Минимальная визуализация, нечитаемые графики |
| 5-6 | Удовлетворительно | Есть графики/схемы, но базового качества |
| 7-8 | Хорошо | Качественная визуализация, информативные схемы |
| 9-10 | Отлично | Профессиональная визуализация, инфографика, кастомные схемы |

### 5c. Читаемость и дизайн

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Нечитаемый текст, хаос в оформлении |
| 3-4 | Не зачтено | Мелкий текст, перегруженные слайды |
| 5-6 | Удовлетворительно | Читаемо, но без внимания к дизайну |
| 7-8 | Хорошо | Чистый дизайн, хорошая типографика, единый стиль |
| 9-10 | Отлично | Профессиональный дизайн, брендированные элементы |

### 5d. Объём и баланс

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | 1-2 слайда или 50+ слайдов без содержания |
| 3-4 | Не зачтено | Слишком мало или слишком много, дисбаланс |
| 5-6 | Удовлетворительно | Адекватный объём, но неравномерное распределение |
| 7-8 | Хорошо | Оптимальный объём, все разделы сбалансированы |
| 9-10 | Отлично | Идеальный баланс: каждый слайд несёт ценность, нет лишнего |

## Формула итоговой оценки (Версия B)

```
Оценка К1 = среднее(1a, 1b, 1c, 1d)
Оценка К2 = среднее(2a, 2b, 2c, 2d)
Оценка К3 = среднее(3a, 3b, 3c, 3d)
Оценка К4 = среднее(4a, 4b, 4c, 4d)
Оценка К5 = среднее(5a, 5b, 5c, 5d)

Итоговая = К1 * 0.25 + К2 * 0.20 + К3 * 0.20 + К4 * 0.20 + К5 * 0.15
```

## Правило зачёта

- **Зачтено:** Итоговая оценка >= 4.0 **И** каждый из критериев (К1-К5) >= 3.0
- **Не зачтено:** Итоговая оценка < 4.0 **ИЛИ** любой из критериев < 3.0

## Отличия от Версии A

| Аспект | Версия A | Версия B |
|--------|----------|----------|
| Критерии | 3 | 5 |
| Подаспекты | 12 | 20 |
| Аналитическая глубина | Не оценивается отдельно | Отдельный критерий (20%) |
| Качество презентации | Не оценивается | Отдельный критерий (15%) |
| Лучше подходит для | Типовых ДЗ, быстрой оценки | Итоговых работ, детальной дифференциации |
| Время оценки | ~5-7 мин на работу | ~10-15 мин на работу |

## Когда использовать Версию B

- Итоговые / курсовые работы
- Когда нужно дифференцировать сильные работы (в Версии A все сильные получают похожие оценки)
- Когда важна обратная связь по качеству презентации
- Когда аналитическая глубина — ключевое требование задания
