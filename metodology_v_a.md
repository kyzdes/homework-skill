# Методология оценки: Версия A (3 критерия)

## Общая информация

- **Курс:** Продуктовый и диджитал-маркетинг
- **Версия:** A — 3 критерия, 12 подаспектов
- **Шкала:** 1-10 по каждому подаспекту
- **Итоговая оценка:** взвешенная средняя по 3 критериям
- **Порог "не зачтено":** средний балл < 4.0 ИЛИ любой из 3 критериев < 3

## Критерии и веса

| # | Критерий | Вес | Подаспекты |
|---|----------|-----|------------|
| К1 | Полнота раскрытия | 40% | 4 подаспекта |
| К2 | Согласованность | 30% | 4 подаспекта |
| К3 | Применение знаний с лекций | 30% | 4 подаспекта |

---

## К1. Полнота раскрытия (40%)

Оценивает, насколько полно студент раскрыл каждый пункт задания.

### 1a. Бизнес-цель
**Что оценивается:** Чёткая формулировка бизнес-цели по SMART, привязка к реальной бизнес-метрике компании.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Бизнес-цель отсутствует или сформулирована бессмысленно |
| 3-4 | Не зачтено | Цель расплывчата ("увеличить продажи"), не SMART, не привязана к реальности |
| 5-6 | Удовлетворительно | Есть цель, частично соответствует SMART (например, есть конкретика, но нет сроков) |
| 7-8 | Хорошо | SMART-цель, конкретная, с измеримыми показателями и сроками |
| 9-10 | Отлично | SMART-цель + привязка к реальной бизнес-метрике компании + обоснование выбора цели |

### 1b. Декомпозиция до маркетинговых целей
**Что оценивается:** Логика разбивки бизнес-цели на маркетинговые цели, измеримость каждой цели.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Декомпозиция отсутствует, маркетинговые цели не определены |
| 3-4 | Не зачтено | 1 маркетинговая цель без логической связи с бизнес-целью |
| 5-6 | Удовлетворительно | 2+ маркетинговых цели, но логика декомпозиции неполная или натянутая |
| 7-8 | Хорошо | Логичная декомпозиция, 3+ измеримых маркетинговых целей |
| 9-10 | Отлично | Полная декомпозиция с обоснованием каждой связи, все цели измеримы и SMART |

### 1c. Каналы и подканалы
**Что оценивается:** Классификация ATL/BTL/TTL, конкретные подканалы для каждого канала.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Каналы не указаны |
| 3-4 | Не зачтено | 1-2 канала без классификации ATL/BTL/TTL, без подканалов |
| 5-6 | Удовлетворительно | 3+ каналов, частичная классификация, некоторые подканалы указаны |
| 7-8 | Хорошо | Полная классификация ATL/BTL/TTL, конкретные подканалы для каждого канала |
| 9-10 | Отлично | Исчерпывающий анализ каналов с обоснованием выбора, релевантные подканалы, альтернативы |

### 1d. Цели подканалов
**Что оценивается:** Уникальная цель и конкретная метрика для каждого подканала.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Цели подканалов не указаны |
| 3-4 | Не зачтено | Общие цели без метрик ("привлечь клиентов"), одинаковые для всех подканалов |
| 5-6 | Удовлетворительно | Частичные цели и метрики, не для всех подканалов |
| 7-8 | Хорошо | Уникальная цель + метрика для каждого подканала |
| 9-10 | Отлично | Уникальные цели + конкретные метрики + целевые значения/бенчмарки для каждого |

## К2. Согласованность (30%)

Оценивает логическую связь между всеми элементами работы.

### 2a. Бизнес-цель → маркетинговые цели
**Что оценивается:** Маркетинговые цели логически вытекают из бизнес-цели.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Маркетинговые цели не связаны с бизнес-целью |
| 3-4 | Не зачтено | Слабая связь, натянутая логика |
| 5-6 | Удовлетворительно | Связь прослеживается, но с логическими пробелами |
| 7-8 | Хорошо | Чёткая логическая связь, маркетинговые цели обоснованно вытекают из бизнес-цели |
| 9-10 | Отлично | Безупречная логическая цепочка с обоснованием каждого перехода |

### 2b. Маркетинговые цели → каналы
**Что оценивается:** Выбор каналов обоснован маркетинговыми целями.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Каналы выбраны случайно, не связаны с маркетинговыми целями |
| 3-4 | Не зачтено | Частичное соответствие, большинство каналов не обосновано |
| 5-6 | Удовлетворительно | Большинство каналов обосновано, но есть нерелевантные |
| 7-8 | Хорошо | Все каналы логично обоснованы маркетинговыми целями |
| 9-10 | Отлично | Каждый канал обоснован + рассмотрены и отвергнуты альтернативы |

### 2c. Каналы → цели подканалов
**Что оценивается:** Цели подканалов отвечают на вопрос "зачем именно этот подканал".

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Полное рассогласование между каналами и целями подканалов |
| 3-4 | Не зачтено | Частичное соответствие, многие цели не релевантны каналам |
| 5-6 | Удовлетворительно | В целом согласовано, но есть нестыковки |
| 7-8 | Хорошо | Полное соответствие, каждая цель подканала работает на цель канала |
| 9-10 | Отлично | Каскадная связь прослеживается сквозь все уровни: бизнес → маркетинг → канал → подканал |

### 2d. Внутренняя непротиворечивость
**Что оценивается:** Отсутствие противоречий между любыми элементами работы.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Явные противоречия (например, цель — снижение стоимости, а каналы — премиальные) |
| 3-4 | Не зачтено | Несколько заметных противоречий |
| 5-6 | Удовлетворительно | Мелкие несогласованности, не критичные |
| 7-8 | Хорошо | Работа в целом согласована, противоречий нет |
| 9-10 | Отлично | Полностью непротиворечивая, целостная система |

## К3. Применение знаний с лекций (30%)

Оценивает, насколько студент применил знания, полученные на лекциях 1 и 2.

### 3a. Фреймворки
**Что оценивается:** Применение маркетинговых фреймворков (4P/7P, ATL/BTL/TTL, SMART, AIDA, воронка продаж и др.)

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Фреймворки не использованы |
| 3-4 | Не зачтено | Упоминание 1 фреймворка без реального применения |
| 5-6 | Удовлетворительно | 1-2 фреймворка применены, но поверхностно |
| 7-8 | Хорошо | 3+ фреймворка применены корректно и к месту |
| 9-10 | Отлично | Глубокое применение всех релевантных фреймворков, демонстрация понимания их взаимосвязей |

### 3b. Метрики
**Что оценивается:** Использование маркетинговых метрик (CTR, VTR, CR, BR, CPA, CPM, Reach, ER, ROAS и др.)

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Метрики полностью отсутствуют |
| 3-4 | Не зачтено | 1-2 метрики упомянуты без контекста |
| 5-6 | Удовлетворительно | Метрики указаны для части каналов, но без целевых значений |
| 7-8 | Хорошо | Релевантные метрики для всех каналов/подканалов |
| 9-10 | Отлично | Метрики + целевые значения + отраслевые бенчмарки + обоснование выбора метрик |

### 3c. Диджитал-концепции
**Что оценивается:** Понимание и применение диджитал-маркетинга (контекстная реклама, таргетированная реклама, SMM, SEO, email-маркетинг, CPA-сети, инфлюенсер-маркетинг и др.)

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Диджитал-концепции не использованы |
| 3-4 | Не зачтено | Поверхностное упоминание 1-2 концепций |
| 5-6 | Удовлетворительно | Базовое понимание диджитал-каналов, но без глубины |
| 7-8 | Хорошо | Хорошее понимание и корректное применение диджитал-концепций |
| 9-10 | Отлично | Глубокое понимание + современные тренды + специфика для выбранного бренда |

### 3d. Корректность терминологии
**Что оценивается:** Правильное использование маркетинговых терминов.

| Балл | Уровень | Описание |
|------|---------|----------|
| 1-2 | Критический провал | Грубые ошибки в терминологии, путаница понятий |
| 3-4 | Не зачтено | Систематические неточности, термины используются неверно |
| 5-6 | Удовлетворительно | Отдельные неточности, но в целом понимание есть |
| 7-8 | Хорошо | Корректная терминология, единичные неточности |
| 9-10 | Отлично | Профессиональный уровень терминологии, точные формулировки |

## Формула итоговой оценки

```
Оценка К1 = среднее(1a, 1b, 1c, 1d)
Оценка К2 = среднее(2a, 2b, 2c, 2d)
Оценка К3 = среднее(3a, 3b, 3c, 3d)

Итоговая = К1 * 0.4 + К2 * 0.3 + К3 * 0.3
```

## Правило зачёта

- **Зачтено:** Итоговая оценка >= 4.0 **И** каждый из критериев (К1, К2, К3) >= 3.0
- **Не зачтено:** Итоговая оценка < 4.0 **ИЛИ** любой из критериев < 3.0

## Ключевые концепции из лекций

> Этот раздел заполняется автоматически при анализе конкретных лекций в рамках skill workflow (Шаг 2).
> Ниже — базовый набор концепций, который дополняется при каждом запуске.

### Фреймворки
- 4P (Product, Price, Place, Promotion) / 7P (+ People, Process, Physical Evidence)
- ATL / BTL / TTL — классификация каналов
- SMART — формулировка целей
- AIDA — модель воронки (Attention, Interest, Desire, Action)
- Маркетинговая воронка (Awareness → Consideration → Conversion → Retention)

### Метрики
- CTR (Click-Through Rate)
- VTR (View-Through Rate)
- CR (Conversion Rate)
- BR (Bounce Rate)
- CPA (Cost Per Acquisition)
- CPM (Cost Per Mille)
- Reach
- ER (Engagement Rate)
- ROAS (Return On Ad Spend)
- LTV (Lifetime Value)

### Диджитал-каналы и концепции
- Контекстная реклама (Яндекс.Директ, Google Ads)
- Таргетированная реклама (VK Ads, myTarget)
- SMM (Social Media Marketing)
- SEO (Search Engine Optimization)
- Email-маркетинг
- CPA-сети
- Инфлюенсер-маркетинг
- Programmatic
- Ретаргетинг / ремаркетинг
